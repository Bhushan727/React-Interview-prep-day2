Q1. What is Difference between react and react native ? Which one is library ?

Ans. React which is also known as react js is a javascript library which is used to build a single page Web Application. it is developed by facebook and gained a huge fanbase in a short span of time.
whereas, react native is a javascript framework which is specially used to develop mobile native apps. it is also created by facebook and it uses reusable components and created a huge fanbase .
react is most commonly used front end technology is developed and still managed by facebook but its not a framework is actually a javascript library. whereas react native is framework which is for ios and android mobile development.
